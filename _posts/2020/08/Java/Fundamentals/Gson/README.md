## Gson
-----

### 1. 使用Gson

使用Gson的首要类是Gson类，你可以仅仅通过new Gson()的方式创建它。

你也可以通过GsonBuilder类去创建Gson实例，这个类允许你进行一系列配置，例如版本控制等等。

Gson实例不会保存任何进行Json操作时的状态。因此，你可以自由的服用相同的Gson对象进行诸多的Json序列化和反序列化操作。
